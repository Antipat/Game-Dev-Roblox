# Game-Dev-Roblox
Коды и локации для книги   


В связи с тем что в РФ и Австралии поломался Roblox есть готовое решение - использовать локальный сервер для перенаправления трафика.
Проект носит название zapret-discord-youtube (https://github.com/Flowseal/zapret-discord-youtube)

Он содержит набор серверов упакованных в формате bat для запуска на Windows
